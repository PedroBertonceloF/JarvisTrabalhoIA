# Domain Glossary

- **Agenda**: The central calendar system that contains all scheduled items. It holds both Eventos and Tarefas.
- **Evento (Event)**: A time-bound occurrence in the Agenda (e.g., classes, exams). It is the base concept for scheduled items.
- **Tarefa (Task)**: A specific type of Evento (a subclass) that represents an actionable item with a completion state (e.g., pending, completed).
- **Plano de Estudos (Study Plan)**: A dynamically generated proposal combining Eventos, Tarefas, and Materiais. It exists initially as a text/view response. It is only converted into actual Tarefas in the Agenda upon explicit user request or interaction (e.g., clicking a button).
- **Material**: A single ingested document (e.g., a PDF or TXT file) used as a knowledge source for the RAG system.
- **Disciplina**: A logical grouping or category that contains a collection of Materiais (not a subclass of Material).
- **Sessão de Revisão (Review Session)**: An interactive learning session explicitly requested by the user. The user selects a Disciplina and specific Materiais, and the system generates questions to evaluate the user's knowledge (active recall).
- **Recomendação de Revisão (Review Recommendation)**: A proactive suggestion generated by the system, identifying topics or Materiais the user struggled with during past Sessões de Revisão. It integrates with the Plano de Estudos to suggest priority areas for study.
